# Salon Tugba

Production-ready React and Vite website for Salon Tugba in Denizli.

## Tech Stack

- React 19
- Vite 8
- Tailwind CSS 3

## Available Scripts

- `npm run dev` starts the local development server
- `npm run build` creates the production build in `dist`
- `npm run lint` runs ESLint checks
- `npm run preview` serves the production build locally

## Project Notes

- Main site content is managed in `src/data/siteContent.js`
- Static media is served from `public/`
- The site is optimized for mobile-first browsing and WhatsApp-based booking

## Deployment

1. Install dependencies with `npm install`
2. Build the app with `npm run build`
3. Deploy the generated `dist/` folder to your hosting platform
